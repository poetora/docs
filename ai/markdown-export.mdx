与 HTML 相比，Markdown 提供的结构化文本更便于 AI 工具高效处理，从而实现更高的回答准确性、更快的处理速度，以及更少的 token 消耗。

Poetora 会自动生成针对 AI 工具和外部 integrations 优化的页面 Markdown 版本。

<div id="md-url-extension">
  ## .md URL 后缀
</div>

在任意页面的 URL 末尾添加 `.md`，即可查看其 Markdown 版本。

<PreviewButton href="https://poetora.com/docs/ai/markdown-export.md">以 Markdown 打开此页面</PreviewButton>

## 键盘快捷键

按 <kbd>Command</kbd> + <kbd>C</kbd>（在 Windows 上为 <kbd>Ctrl</kbd> + <kbd>C</kbd>）将页面以 Markdown 格式复制到剪贴板。
