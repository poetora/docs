## 浏览你的文件

使用侧边栏文件资源管理器浏览你的文档文件。点击任何文件即可在编辑器中打开。

## 创建新页面

<Steps>
  <Step title="创建新文件">
    在文件资源管理器侧边栏中选择**创建新文件**图标。
    <img
      src="/images/editor/create-new-file-light.png"
      alt="Poetora 编辑器创建新文件"
    />
  </Step>
  <Step title="输入文件名">
    文件名必须以 `.mdx`
    结尾。例如，`api-reference.mdx` 或
    `getting-started.mdx`。
  </Step>
  <Step title="创建并打开">
    按 <kbd>Enter</kbd>{' '}
    创建文件。它将在编辑器中自动打开，准备添加内容。
  </Step>
</Steps>

## 组织你的导航

通过编辑 `docs.json` 文件将新页面添加到你的站点导航中。

<Steps>
  <Step title="打开 docs.json">
    使用文件资源管理器在根目录中找到并打开
    `docs.json` 文件。
  </Step>
  <Step title="定位导航部分">
    在 `docs.json`
    文件中找到你想要添加新页面的位置。
  </Step>
  <Step title="添加你的页面">
    在相应组的 `pages`
    数组中插入你的页面文件名（不包含 `.mdx`
    扩展名）。
  </Step>
</Steps>

### 示例：添加主题页面

```json title="将主题页面添加到配置组" highlight="18"
{
  "navigation": {
    "groups": [
      {
        "group": "快速开始",
        "pages": [
          "index",
          "quickstart",
          "installation"
        ]
      },
      {
        "group": "配置",
        "pages": [
          "settings",
          "account-types",
          "dashboard",
          "themes"
        ]
      }
    ]
  }
}
```

## 编辑内容

在编辑器中使用可视化模式或 Markdown 模式对页面进行更改。

在可视化模式下，按 <kbd>/</kbd> 打开组件菜单。添加内容块、提示框、代码块和其他组件来自定义你的文档。

<Frame>
  <img
    src="/images/editor/slash-command-menu-light.png"
    alt="Poetora 编辑器组件菜单"
  />
</Frame>

在 Markdown 模式下，你可以直接编辑页面的 `MDX`。当你需要以下功能时，这可能会有所帮助：

- 设置特定的组件属性
- 使用复杂的嵌套组件
- 从其他源复制和粘贴 `MDX` 内容
