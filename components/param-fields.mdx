## 组件属性

### ParamField 属性

<ParamField path="path" type="string">
  参数路径或名称。用于定义 URL 路径中的参数，如 `/users/{id}` 中的 `id`。
</ParamField>

<ParamField query="query" type="string">
  查询参数名称（与 path 二选一）。用于定义 URL 查询字符串中的参数，如 `?limit=10` 中的 `limit`。
</ParamField>

<ParamField body="body" type="string">
  请求体参数名称（与 path 二选一）。用于定义 POST、PUT 等请求的请求体中的参数。
</ParamField>

<ParamField header="header" type="string">
  请求头参数名称（与 path 二选一）。用于定义 HTTP 请求头中的参数，如 `Authorization`。
</ParamField>

<ParamField path="type" type="string" required>
  参数类型。支持基础类型（string、number、boolean、object）和数组类型（string[]、number[]等）。
</ParamField>

<ParamField path="required" type="boolean" default="false">
  是否为必需参数。设置为 `true` 时，该参数在 API 调用中必须提供。
</ParamField>

<ParamField path="default" type="string">
  参数的默认值。当请求中未提供该参数时使用的默认值。
</ParamField>

<ParamField path="deprecated" type="boolean" default="false">
  是否已废弃。设置为 `true` 时，参数会显示废弃标识，提醒开发者使用替代方案。
</ParamField>

### 支持的参数类型

**基础类型：**

- `string` - 字符串
- `number` - 数字
- `boolean` - 布尔值
- `object` - 对象
- `array` - 数组

**复合类型：**

- `string[]` - 字符串数组
- `number[]` - 数字数组
- `User` - 自定义对象类型
- `User[]` - 自定义对象数组
