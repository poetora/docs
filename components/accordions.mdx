## 组件属性

### Accordion 属性

<ParamField path="title" type="string" required>
  手风琴的标题文本。显示在可点击的标题栏中，用于标识内容区域的主题。
</ParamField>

<ParamField path="description" type="string">
  标题下方的描述文本。提供关于手风琴内容的简要说明，显示在标题下方。
</ParamField>

<ParamField path="icon" type="string">
  标题前显示的图标。支持 Font Awesome 和 Lucide 图标库中的图标名称，如 `"chevron-down"`、`"info"` 等。
</ParamField>

<ParamField path="defaultOpen" type="boolean" default="false">
  是否默认展开。设置为 `true` 时，手风琴在页面加载时将默认处于展开状态。
</ParamField>

## 最佳实践

### 内容组织

- **逻辑分组** - 将相关内容放在同一个手风琴组中
- **标题清晰** - 使用简洁明了的标题
- **内容适量** - 每个手风琴项的内容不宜过长
- **层级合理** - 避免过度嵌套，保持结构清晰

### 交互体验

- **重要内容优先** - 将最重要的内容设置为默认展开
- **图标辅助** - 使用图标帮助用户快速识别内容类型
- **描述补充** - 为复杂标题提供简短描述
- **响应式设计** - 确保在移动设备上也有良好体验

### 性能考虑

- **内容懒加载** - 对于大量内容可以考虑懒加载
- **图片优化** - 压缩手风琴中的图片大小
- **嵌套限制** - 避免在手风琴中嵌套过多复杂组件

## 可访问性

手风琴组件支持完整的可访问性功能：

- **键盘导航** - 支持 Tab、Enter、Space 等键盘操作
- **屏幕阅读器** - 提供适当的 ARIA 标签和语义
- **焦点管理** - 清晰的焦点指示和合理的焦点顺序
- **状态提示** - 明确指示展开/折叠状态

### 键盘快捷键

| 按键              | 功能                        |
| ----------------- | --------------------------- |
| `Tab`             | 在手风琴项之间导航          |
| `Enter` / `Space` | 展开/折叠当前手风琴项       |
| `↑` / `↓`         | 在手风琴组中上下移动        |
| `Home` / `End`    | 跳到第一个/最后一个手风琴项 |
