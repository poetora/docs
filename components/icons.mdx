# 图标组件

图标组件允许你在文档中使用来自 Font Awesome 和 Lucide 图标库的图标，为文档增加视觉元素和改善用户体验。

## 基本用法

使用 `<Icon>` 组件插入图标：

<Icon icon="flag" size={32} />

```mdx 基本图标示例
<Icon icon="flag" size={32} />
```

## 自定义图标

### 使用 SVG 图标

你也可以使用自定义的 SVG 图标：

```mdx SVG 图标示例
<Icon icon="/icons/custom-logo.svg" size={32} />
```

### 使用外部图标 URL

```mdx 外部图标示例
<Icon
  icon="https://example.com/icons/custom-icon.svg"
  size={24}
/>
```

## 组件属性

<ParamField path="icon" type="string" required>
  图标名称、SVG 文件路径或外部 URL。支持：
  - **Font Awesome 图标**: 如 `"rocket"`、`"user"`、`"settings"`
  - **Lucide 图标**: 如 `"check"`、`"arrow-right"`、`"heart"`
  - **SVG 文件路径**: 如 `"/icons/custom-icon.svg"`
  - **外部 URL**: 如 `"https://example.com/icon.svg"`
</ParamField>

<ParamField path="iconType" type="string" default="regular">
  Font Awesome 图标类型。仅对 Font Awesome 图标有效，支持以下类型：
  - `regular` - 常规样式
  - `solid` - 实心样式
  - `light` - 轻量样式
  - `thin` - 超细样式
  - `sharp-solid` - 尖锐实心样式
  - `duotone` - 双色样式
  - `brands` - 品牌图标样式
</ParamField>

<ParamField path="color" type="string">
  图标颜色。使用十六进制颜色代码定义图标颜色，如 `"#FF5733"`、`"#3B82F6"`、`"#10B981"` 等。
</ParamField>

<ParamField path="size" type="number" default="24">
  图标大小。以像素为单位设置图标的宽度和高度，常用值：16、20、24、32、48。
</ParamField>

## 图标库参考

### Font Awesome 图标

Font Awesome 提供了数千个高质量图标，访问 [Font Awesome 官网](https://fontawesome.com/icons) 浏览完整图标库。

### Lucide 图标

Lucide 是一个现代化的图标库，提供简洁美观的线条图标，访问 [Lucide 官网](https://lucide.dev/icons) 查看所有可用图标。

## 最佳实践

1. **保持一致性** - 在同一文档中使用相同的图标风格和大小
2. **语义化使用** - 选择与内容含义相符的图标
3. **适度使用** - 避免过度使用图标，保持文档的简洁性
4. **无障碍考虑** - 确保图标不是传达信息的唯一方式
5. **加载性能** - 对于大量图标，考虑使用图标字体而非单独的 SVG 文件

图标组件是增强文档视觉效果和用户体验的重要工具。合理使用图标可以让你的文档更加专业、易读和用户友好。
