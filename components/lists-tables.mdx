## 列表使用

列表遵循官方 [Markdown 语法](https://www.markdownguide.org/basic-syntax/#lists-1) 规范。

### 有序列表

要创建有序列表，在列表项前添加数字和句点：

1. 第一项
2. 第二项
3. 第三项
4. 第四项

```mdx
1. 第一项
2. 第二项
3. 第三项
4. 第四项
```

### 无序列表

要创建无序列表，在列表项前添加破折号 (`-`)、星号 (`*`) 或加号 (`+`)：

- 第一项
- 第二项
- 第三项
- 第四项

```mdx
- 第一项
- 第二项
- 第三项
- 第四项
```

### 嵌套列表

缩进列表项来创建嵌套结构：

- 第一项
- 第二项
  - 附加项目
  - 附加项目
- 第三项

```mdx
- 第一项
- 第二项
  - 附加项目
  - 附加项目
- 第三项
```

## 表格使用

表格遵循官方 [Markdown 语法](https://www.markdownguide.org/extended-syntax/#tables) 规范。

要添加表格，使用三个或更多连字符 (`---`) 创建每列的标题，使用竖线 (`|`) 分隔每列。为了兼容性，还应该在行的两端添加竖线。

| 属性 | 描述               |
| ---- | ------------------ |
| 姓名 | 用户的完整姓名     |
| 年龄 | 用户报告的年龄     |
| 加入 | 用户是否已加入社区 |

```mdx
| 属性 | 描述               |
| ---- | ------------------ |
| 姓名 | 用户的完整姓名     |
| 年龄 | 用户报告的年龄     |
| 加入 | 用户是否已加入社区 |
```
