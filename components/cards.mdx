## 组件属性

<ParamField path="title" type="string" required>
  卡片标题。显示在卡片顶部的主要标题文本，必填项。
</ParamField>

<ParamField path="icon" type="string">
  图标名称。支持 Font Awesome 和 Lucide 图标库中的图标名称，如 `"rocket"`、`"settings"` 等。
</ParamField>

<ParamField path="color" type="string">
  图标颜色。使用十六进制颜色代码定义图标颜色，如 `"#FF5733"`、`"#3B82F6"` 等。
</ParamField>

<ParamField path="href" type="string">
  链接地址。点击卡片时跳转的 URL，可以是相对路径或绝对路径。
</ParamField>

<ParamField path="horizontal" type="boolean" default="false">
  是否使用横向布局。设置为 `true` 时，卡片内容水平排列而非垂直排列。
</ParamField>

<ParamField path="img" type="string">
  图片 URL 或路径。显示在卡片中的图片资源，支持相对路径和绝对 URL。
</ParamField>

<ParamField path="cta" type="string">
  自定义链接文本。自定义卡片底部的行动按钮文本，默认为 "了解更多"。
</ParamField>

<ParamField path="arrow" type="boolean" default="false">
  是否显示箭头图标。设置为 `true` 时在卡片右侧显示指向箭头。
</ParamField>

## 最佳实践

### 内容组织

- **标题简洁** - 使用简短而描述性的标题
- **内容聚焦** - 每个卡片专注于一个主题
- **层次清晰** - 重要信息优先显示
- **行动导向** - 明确告诉用户下一步做什么

### 视觉设计

- **图标一致** - 使用统一风格的图标
- **颜色搭配** - 避免过多颜色造成视觉混乱
- **间距均匀** - 保持卡片之间适当的间距
- **对齐整齐** - 确保卡片网格对齐

### 响应式布局

- **移动优先** - 考虑移动设备的显示效果
- **自适应列数** - 根据屏幕大小调整列数
- **内容简化** - 在小屏幕上突出核心信息
- **触控友好** - 确保足够的点击区域

### 性能优化

- **图片优化** - 压缩图片并使用适当格式
- **懒加载** - 对大量卡片实施懒加载
- **内容精简** - 避免在卡片中放置过多内容
- **链接预加载** - 对重要页面实施预加载

## 可访问性

卡片组件支持完整的可访问性功能：

- **键盘导航** - 支持 Tab 键在卡片间导航
- **屏幕阅读器** - 提供清晰的内容描述
- **焦点指示** - 清晰的焦点高亮效果
- **语义化标签** - 使用适当的 HTML 语义

### 键盘操作

| 按键    | 功能                   |
| ------- | ---------------------- |
| `Tab`   | 在可点击的卡片间导航   |
| `Enter` | 激活当前聚焦的卡片链接 |
| `Space` | 激活当前聚焦的卡片链接 |
