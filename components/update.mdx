## 使用说明

```mdx Update example
<Update label="2025-11-10" description="v1.0.0" tags={["Poetora"]}>
  这是一个带有标签、说明和标记的更新。
</Update>
```

使用多个 `Update` 组件创建[更新日志](/create/update)。

## 组件属性

<ParamField path="label" type="string" required>
  更新标签。显示在左侧的主要标识文本，同时会自动创建锚点链接供页面内导航使用。
</ParamField>

<ParamField path="description" type="string">
  更新描述。显示在标签下方的补充说明文字，用于提供更多上下文信息。
</ParamField>

<ParamField path="tags" type="string[]">
  标签列表。用于分类更新内容的标签数组，这些标签会显示为右侧面板的过滤器，方便用户按类型筛选更新。
</ParamField>
