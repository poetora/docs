# 列表和表格

在文档中展示信息列表和表格，提供清晰的数据组织方式。

## 列表

列表遵循官方的 [Markdown 语法](https://www.markdownguide.org/basic-syntax/#lists-1)。

### 有序列表

要创建有序列表，在列表项前添加数字和句点。

1. 第一项
2. 第二项
3. 第三项
4. 第四项

```mdx
1. 第一项
2. 第二项
3. 第三项
4. 第四项
```

### 无序列表

要创建无序列表，在列表项前添加破折号 (`-`)、星号 (`*`) 或加号 (`+`)。

- 第一项
- 第二项
- 第三项
- 第四项

```mdx
- 第一项
- 第二项
- 第三项
- 第四项
```

### 嵌套列表

缩进列表项来嵌套它们。

- 第一项
- 第二项
  - 附加项目
  - 附加项目
- 第三项

```mdx
- 第一项
- 第二项
  - 附加项目
  - 附加项目
- 第三项
```

## 表格

表格遵循官方的 [Markdown 语法](https://www.markdownguide.org/extended-syntax/#tables)。

要添加表格，使用三个或更多连字符 (`---`) 创建每列的标题，并使用管道符 (`|`) 分隔每列。为了兼容性，你还应该在行的两端添加管道符。

| 属性   | 描述               |
| ------ | ------------------ |
| 姓名   | 用户的完整姓名     |
| 年龄   | 报告的年龄         |
| 已加入 | 用户是否已加入社区 |

```mdx
| 属性   | 描述               |
| ------ | ------------------ |
| 姓名   | 用户的完整姓名     |
| 年龄   | 报告的年龄         |
| 已加入 | 用户是否已加入社区 |
```

### 表格对齐

你可以通过在标题行的连字符左侧、右侧或两侧添加冒号 (`:`) 来对齐列中的文本。

| 左对齐         |   居中对齐   |         右对齐 |
| :------------- | :----------: | -------------: |
| 这列是左对齐的 | 这列是居中的 | 这列是右对齐的 |
| 左             |      中      |             右 |

```mdx
| 左对齐         |   居中对齐   |         右对齐 |
| :------------- | :----------: | -------------: |
| 这列是左对齐的 | 这列是居中的 | 这列是右对齐的 |
| 左             |      中      |             右 |
```

## 最佳实践

### 列表使用建议

1. **保持一致性** - 在同一文档中使用一致的列表样式
2. **适当嵌套** - 避免过深的嵌套层级
3. **清晰的层次** - 使用不同的列表符号区分层级
4. **并行结构** - 确保同级列表项的语法结构一致

### 表格使用建议

1. **简洁明了** - 保持表格内容简洁，避免过长的文本
2. **合理列宽** - 确保所有列都有足够的空间
3. **响应式设计** - 为移动设备优化表格显示
4. **数据对齐** - 数字右对齐，文本左对齐，标题居中
5. **交替行色** - 对于长表格，考虑使用交替行背景色

### 可访问性考虑

1. **表格标题** - 为复杂表格添加标题和摘要
2. **列标题** - 确保每列都有清晰的标题
3. **逻辑顺序** - 按逻辑顺序组织表格数据
4. **屏幕阅读器** - 考虑屏幕阅读器用户的体验

通过合理使用列表和表格，可以让文档中的信息更加清晰、有条理，提升用户阅读体验。
